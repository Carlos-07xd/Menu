Router Initial Configuration

  • 1. Connect the Router:
    • Plug the router into a power outlet.
    • Connect the internet cable to the router's WAN or Internet port.
    • Connect your computer to the router via an Ethernet cable or Wi-Fi.
  • 2. Access the Configuration Interface:
    • Open a web browser and enter the router's IP address (usually 192.168.0.1 or 192.168.1.1).
    • Enter the default credentials (username: admin, password: admin or password).
  • 3. Configure Internet Access:
    • Select the connection type provided by your Internet Service Provider (DHCP, PPPoE, etc.).
    • Enter any required information (such as username and password for PPPoE).
  • 4. Configure Wi-Fi:
    • Change the wireless network name (SSID).
    • Set a secure password using WPA2 or WPA3 encryption.
  • 5. Change Admin Credentials:
    • Change the default username and password for router access.
  • 6. Update the Firmware:
    • Check for available updates and apply them.
  • 7. Save and Restart:
    • Save all changes and restart the router to apply the configuration.
Note: Refer to your router's manual for specific information. If you have questions, contact your Internet Service Provider.
