Configure Wi-Fi on TP-Link Archer C50

  • 1. Connect the Router:
    • Plug the router into the power outlet.
    • Connect the internet cable to the router's WAN port (usually blue).
    • Connect your computer to the router via an Ethernet cable or Wi-Fi.
      The default network name and password are on the router's label.
  • 2. Access the Router Settings:
    • Open a web browser and enter the router's IP address (192.168.0.1 or 192.168.1.1).
    • Log in using the default credentials:
      • Username: admin
      • Password: admin
  • 3. Configure the Wi-Fi Network:
    • In the menu, select Wireless.
    • Set the following options for the 2.4 GHz and 5 GHz bands:
      • Network Name (SSID): Customize the Wi-Fi network name.
      • Operation Mode: Use the default setting ("11bgn mixed" for 2.4 GHz, "11ac/n mixed" for 5 GHz).
      • Channel: Leave it on Auto or choose manually if interference occurs.
      • Bandwidth: Select 20 MHz or 20/40 MHz for 2.4 GHz and 40 MHz or 80 MHz for 5 GHz.
    • In Wireless Security, select:
      • WPA2-PSK or WPA3-PSK as the security protocol.
      • Enter a secure password.
  • 4. Save and Restart:
    • Click Save after each configuration.
    • Restart the router from System Tools > Reboot.
  • 5. Connect Your Devices:
    • Search for the new Wi-Fi network name and connect using the configured password.
Note: If you encounter issues, consult the TP-Link Archer C50 manual or contact TP-Link support.
